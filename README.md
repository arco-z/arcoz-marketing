# 🦷 아르코즈(Arcoz) 치과 콘텐츠 마케팅 시스템

웹 에이전시(Arcoz)의 기술력과 데이터 분석 역량을 결합하여 구축한 **'성과 측정형 로컬 병원 마케팅'** 전문 시스템입니다. 
단순한 상위 노출이나 무의미한 블로그 포스팅이 아닌, 타겟 환자의 본질적인 불안감을 해소하고 내원 전환율을 극대화하는 것을 목표로 합니다.

## 🎯 핵심 가치 (Core Value)

1. **데이터 기반 타겟팅:** 검색량은 적지만 전환율(예약)이 압도적으로 높은 **'고관여 롱테일 키워드'** 발굴.
2. **AI 자동화 파이프라인:** Gemini API와 n8n을 활용하여 블로그 초안, SNS 카드뉴스, 유튜브 숏폼 대본을 원클릭으로 가공하는 **콘텐츠 팩토리(OSMU)** 구축.
3. **초밀착 로컬 퍼널:** 지식iN(권위) ➔ 네이버 플레이스(신뢰) ➔ 카카오톡 챗봇(예약)으로 이어지는 빈틈없는 전환 설계.
4. **의료법 리스크 제로:** 과장 광고 방지 및 환자 후기 우회 작성 등 엄격한 준법 가이드라인 적용.
