# 치과 마케팅 에이전시 런칭: 4주 실행 계획 (Action Plan)

지금까지 세운 기획안을 바탕으로, **실제 에이전시 서비스를 런칭하고 첫 고객을 온보딩하기까지의 4주 완성 스케줄**입니다. 

---

## [1주 차] 파이프라인(인프라) 세팅 및 샘플 확보
가장 먼저 우리가 팔 '무기'를 실제 작동 가능한 상태로 만들어야 합니다.

- [ ] **Task 1.1: 구글 워크스페이스 세팅 (무료 대안)**
  - 구글 드라이브 내 '콘텐츠 검수 보드' 폴더 생성
  - 블로그 본문, 카드뉴스 텍스트, 쇼츠 대본이 담길 구글 문서(Docs) 또는 구글 시트(Sheets) 템플릿 양식 만들기
- [ ] **Task 1.2: n8n ✕ Gemini API 오토메이션 구축 (`06_gemini_n8n_workflow.md` 참고)**
  - 구글 시트 트리거 연결 및 Gemini 1.5 Pro/Flash 프롬프트 세팅
  - 구글 문서(Docs) API 연결하여 결과물 자동 적재 테스트 완료하기
- [ ] **Task 1.3: 에이전시 공용 카카오톡 챗봇 세팅**
  - 치과에 데모로 보여줄 **[사진 상담 챗봇]**과 **[환영 메시지]** 템플릿 임시 생성
- [ ] **Task 1.4: 가상의 포트폴리오(샘플) 1건 제작**
  - 테스트용 키워드("강남 임플란트 뼈이식 통증")를 잡아 n8n을 돌려보기
  - 추출된 텍스트로 미리캔버스/Canva를 활용해 '인스타 카드뉴스 5장' 실제 디자인물 1세트 만들어두기

## [2주 차] 세일즈 머티리얼 준비 및 타겟 리스트업
무기가 준비되었으니, 원장님을 설득할 제안서를 다듬고 영업 대상을 뾰족하게 찾습니다.

- [ ] **Task 2.1: 영업 제안서 완성 (`04_sales_proposal.md` 참고)**
  - 1주 차에 만든 '샘플 카드뉴스'와 'n8n 자동화 시연 캡처본'을 제안서 슬라이드에 삽입하여 설득력 극대화
  - 제안서를 PDF로 변환하여 아이패드/노트북에 저장
- [ ] **Task 2.2: 타겟 병원 리스트업 (30곳)**
  - 네이버 지도를 켜고 지역 내 치과 검색 
  - **영업 최적 타겟:** 개원한 지 1~2년 차로 공격적인 마케팅이 필요한 치과, 혹은 블로그를 운영 중이나 한 달 이상 방치된 치과 중심
- [ ] **Task 2.3: 콜드메일 발송 및 전화/방문 컨택**
  - "원장님 병원의 플레이스 및 검색 노출 현황을 무료로 진단해 드립니다"라는 후킹 메시지로 미팅(Call) 잡기

## [3주 차] 1호 고객 미팅 및 계약 (Closing)
- [ ] **Task 3.1: 대면 미팅 및 진단 리포트 브리핑**
  - 우리 웹 에이전시의 강점(데이터 기반, 자동화 퍼널) 어필
- [ ] **Task 3.2: 계약서 작성 (Scope of Work 명확화)**
  - 월 4회(주 1회) 발행 기준, 업무 범위(블로그+카드뉴스+지식iN 업로드 대행) 및 결제일 픽스
  - **(중요)** 병원 내부 CRM(환자 차트)에는 접근하지 않고 외부 채널만 관리한다는 점 명시

## [4주 차] 첫 고객 온보딩 및 파이프라인 1 Cycle 가동
계약 직후 빠르게 성과(초안)를 보여주어 신뢰를 얻는 주간입니다.

- [ ] **Task 4.1: 계정 권한 인수인계 (Onboarding)**
  - 클라이언트의 네이버 플레이스 관리자 권한 부여받기
  - 원장님 의사 면허증으로 네이버 지식iN 'eXpert' 전문가 계정 세팅
  - 카카오비즈니스 채널 관리자 권한 공유받기
- [ ] **Task 4.2: 타겟 키워드 발굴 (`07_keyword_research.md` 참고)**
  - 블랙키위/지식iN을 활용해 첫 달(4주 치) 타겟 롱테일 키워드 4개 뽑기
- [ ] **Task 4.3: 첫 자동화 콘텐츠 퍼블리싱**
  - 구글 시트에 "1주 차 키워드"와 "원장님 메모 3줄" 입력하여 n8n 구동
  - 구글 문서(Docs)에 떨어진 결과물을 에디터가 검수 및 사진 병합
  - 네이버 블로그 / 지식iN 답변 / 플레이스 소식 탭 / 인스타그램에 **동시 발행** 완료!
