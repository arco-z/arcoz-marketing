# 05. AI 기반 자동화 콘텐츠 생산 파이프라인 (에이전시 초격차 전략)

에이전시가 한 달에 치과 10곳(치과당 월 4건 발행 = 총 40건)을 마크하려면, 100% 사람이 쓰는 방식으로는 인건비(외주 작가비) 때문에 수익성이 떨어집니다. 
따라서 **아이디어 발상 ➔ 초안 작성 ➔ 이미지 가공 ➔ 발행**의 전 과정을 AI와 n8n(자동화 툴)으로 압축하여 원가를 낮추고 퀄리티를 상향 평준화하는 전략입니다.

---

## 1. 파이프라인 4단계 구조 (The AI Factory)

### [STEP 1] 아이디어와 키워드 수집 (Human + Data)
단순한 챗봇 질문이 아닌, 사람들이 진짜 검색하는 데이터를 AI에게 먹여야 합니다.
1. **에이전시 소싱:** 네이버/구글 키워드 도구를 통해 트래픽이 높은 타겟 키워드 4개(예: 임플란트 뼈이식 붓기)를 선정합니다.
2. **원장님 소싱:** 원장님께 가장 성공적이었던 환자 엑스레이 전/후 사진 1장과, 이 시술의 핵심 3문장을 카톡으로 받습니다. (이게 유일한 '인간'의 창작 영역입니다.)

### [STEP 2] 프롬프트 엔지니어링 & 초안 생성 (AI)
LLM(ChatGPT, Claude 등)에 치과 전문 지식과 의료법 필터링 프롬프트를 세팅해 둡니다.
- **System Prompt 세팅 (치과 블로그 전용):**
  > "너는 15년 차 구강악안면외과 전문의 대표원장이야. 글의 목적은 '수술에 대한 두려움을 없애주고 전문성을 어필하여 내원하게 만드는 것'이야. 친근한 말투(~해요)보다는 차분하고 논리적인 평어(~습니다)를 사용해 줘. 그리고 절대 '최고', '부작용 없는' 같은 과장된 단어는 쓰지 마."
- **실행 로직 (n8n API 연동 가능):**
  원장님께 받은 3문장 + 타겟 키워드를 n8n Webhook이나 Slack으로 쏘면 ➔ OpenAI API가 물고 들어가서 H1, H2 태그 구조가 잡힌 1,500자 분량의 'SEO 최적화 블로그 초안'을 1분 만에 뽑아냅니다.

### [STEP 3] 교정 및 인간의 터치 (Human-in-the-loop)
AI가 뱉은 글을 그대로 올리면 '양산형 티'가 납니다. 에이전시 매니저(또는 내부 에디터)가 10분을 투자해 양념을 칩니다.
- **어색한 문맥 수정:** AI 특유의 번역 투(예: "당신은 알아야 합니다")를 자연스러운 한국어로 바꿉니다.
- **실제 사진 병합:** AI는 실제 임상 사진이 없으므로, Step 1에서 원장님께 받은 엑스레이 전/후 사진이나 치과 장비 사진을 문단 사이에 자연스럽게 배치합니다.

### [STEP 4] 원소스 멀티유즈 (OSMU) 자동 가공 (AI + n8n)
블로그 원고가 완성되면, 이 원본을 바탕으로 다른 채널의 콘텐츠를 '버튼 클릭 한 번'으로 찍어냅니다.
- **카드뉴스 가공:** ChatGPT에게 "이 블로그 원고를 인스타그램 카드뉴스 5장 분량으로 요약해 줘. 슬라이드별로 텍스트 분리해."라고 명령 ➔ 나온 텍스트를 Canva나 Figma의 미리 세팅된 병원 템플릿에 붙여넣기만 합니다.
- **지식iN 답변 가공:** 블로그 원고를 바탕으로 관련된 대표 질문 3개를 스스로 만들어내고, 그에 대한 지식iN용 답변 템플릿도 뽑아내게 합니다.
- **숏폼 대본 추출:** "원장님이 스마트폰 앞에서 1분 동안 읽을 유튜브 쇼츠 대본 형식으로 요약해 줘."

## 2. 에이전시의 툴스택 (Tool Stack) 제안

1. **텍스트 생성 (Claude 3.5 Sonnet 또는 GPT-4o):**
   - 특히 의료나 전문적인 논조, 구조화된 글쓰기는 Claude 3.5 Sonnet이 GPT보다 한국어 문맥과 '인간 다운' 뉘앙스를 훨씬 잘 잡아냅니다. (적극 추천)
2. **썸네일 및 카드뉴스 (Canva / Midjourney):**
   - 치과 장비나 깨끗한 치아 이미지 등 스톡 사진이 필요할 때는 Midjourney(AI 이미지)를 활용하여 저작권 문제없이 고퀄리티 소스를 무한 생성합니다.
3. **파이프라인 연결 (n8n):**
   - 에이전시가 잘 다루는 n8n을 활용해 '구글 시트에 키워드/주제 입력 ➔ Gemini API가 초안 작성 ➔ 구글 문서(Google Docs)에 발행 예정 글로 자동 저장'까지 백엔드 자동화를 구축합니다. 에디터는 구글 문서에 들어온 초안만 검수하고 퍼블리싱하면 됩니다.

## 3. 수익률 개선 시뮬레이션
*   **전통적 방식 (외주 프리랜서 고용 시):** 치과 1곳당 월 4건 블로그 대행. 건당 외주비 3~5만 원 ➔ 월 원가 12~20만 원. (에이전시 마진 감소, 퀄리티 편차 큼)
*   **AI 공장 방식 (내부 에디터 1명 + AI):** AI가 초안을 잡으므로 작성 시간 건당 1~2시간에서 15분으로 단축. 내부 직원 1명이 한 달에 치과 10~20곳(월 40~80건) 너끈히 관리 가능. (원가 제로에 수렴, 무한 확장 가능)
